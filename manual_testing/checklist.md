# Чек-лист проверки авторизации, страницы товаров и корзины на Saucedemo (https://www.saucedemo.com/)

## Страница логина
 Поле "Username" принимает ввод текста
 Поле "Password" принимает ввод и скрывает символы (точки)
 При оставлении поля "Password" пустым вылезает сообщение об ошибке "Epic sadface: Password is required"
 При оставлении поля "Username" пустым вылезает сообщение об ошибке "Epic sadface: Username is required"
 При неправильном вводе Л"Username" и "Password" вылезает сообщение об ошибке "Epic sadface: Username and password do not match any user in this service"
 При вводе locked_out_user вылезает сообщение об ошибке "Epic sadface: Sorry, this user has been locked out"
 При заполнении кореектных данных происходит переход на страницу https://www.saucedemo.com/inventory.html

## Страница товаров (Inventory)
Отображается список товаров в виде карточек
У каждого товара есть описание, цена и кнопка "Add to cart"
У каждой карточки при наведении на картинку или название товара они кликабельны, и переносят на отдельную страницу с выбранным товаром
Кнопка "Back to products" на странице с выбранным товаром, возврщает нас обратно ко всему каталогу 
При нажатии кнопки "Add to cart" она добавляет товар в корзину "справа в углу значок корзина, цифра увеличивается на 1" и меняет свое название на "Remove"
При нажатии кнопки "Remove" она убирает товар из корзины "справа в углу значок корзина, цифра уменьшается на 1" и меняет свое название на "Add to cart"
Если убрать все товары из корзины, то корзина будет пустой (не будет красных цифр)
При нажатии кнопки "Фильтрация" выпадает список по которому мы можем фильтровать каталог с товарами (всего 4 критерия фильтрации)
При выборе фильтра Name (A to Z) товары идут по алфавиту 
При выборе фильтра Name (Z to A) товары идут по алфавиту с конца
При выборе фильтра Price (low to high) товары идут по возрастанию цены
При выборе фильтра Price (high to low) товары идут по убыванию цены
При нажатии на значок Twitter  переходим на страницу сайта в Twitter
При нажатии на значок Facebook переходим на страницу сайта в Facebook
При нажатии на значок LinkedIn переходим на страницу сайта в LinkedIn

## Корзина
При клике на иконку корзины открывается страница [/cart.html](https://www.saucedemo.com/cart.html)
Из корзины можно удалять товары нажав на кнопку "Remove" (товар удаляется) и справа вверху цифра уменьшается на 1 
При нажатии кнопки "Continue Shopping" нас переносит назад в каталог товаров 
При нажатии кнопки "Checkout" нас переносит на страницу https://www.saucedemo.com/checkout-step-one.html 
При нажатии кнопки "Cancel" нас переносит назад в корзину с товарами 
При оставлении пустых полей "Ferst Name", "Last Name" и "Zip/Postal code" вылезает сообщение об ошибке "Error: First Name is required"
При оставлении пустых полей ""Last Name" и "Zip/Postal code" вылезает сообщение об ошибке "Error: Last Name is required"
При оставлении пустых полей "Ferst Name", "Last Name" вылезает сообщение об ошибке "Error: First Name is required" 
При оставлении пустых полей "Ferst Name", "Zip/Postal code" вылезает сообщение об ошибке "Error: First Name is required" 
При оставлении пустого поля "Zip/Postal code" вылезает сообщение об ошибке "Error: Postal Code is required"
При оставлении пустого поля "Ferst Name" вылезает сообщение об ошибке "Error: First Name is required"
При оставлении пустого поля ""Last Name" вылезает сообщение об ошибке "Error: Last Name is required" 
При заполении всех полей "Ferst Name", "Last Name" и "Zip/Postal code" и нажатии на кнопку "Continue" нас перебрасывает на страницу https://www.saucedemo.com/checkout-step-two.html 
На странице "Оформление заказа" информация о наших товарах, названия добавлених товаров кликабельны и переносят нас на конкретный товар
Кнопка "Cancel" возвращает нас обратно на страницу "Каталога с товарами"
Кнопка "Finish" перебрасывает нас на страницу https://www.saucedemo.com/checkout-complete.html
На старице "Оформление заказа завершено" высвечивается зеленая галочка и информация о том, что заказ оформлен 
Кнопка "Back Home" возарашет нас на страницу с каталогом товаров 
Кнопка "Generate PDF order" скачивает PDF файл с информацией чека о покупке
