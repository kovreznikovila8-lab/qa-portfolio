## TC-001: Успешная авторизация с корректными данными
Предусловия: страница логина открыта (https://www.saucedemo.com/), поля ввода пустые.
Шаги:
1. Ввести "standard_user" в поле "Username".
2. Ввести "secret_sauce" в поле "Password".
3. Нажать кнопку "Login".
Ожидаемый результат: происходит переход на страницу https://www.saucedemo.com/inventory.html, отображается список товаров.
Фактический результат: происходит переход на страницу https://www.saucedemo.com/inventory.html, отображается список товаров.

## TC-009: Добавление товара в корзину
Предусловия: пользователь авторизован (standard_user), находится на странице товаров (https://www.saucedemo.com/inventory.html).
Шаги:
1. Нажать кнопку "Add to cart" у первого товара (например, "Sauce Labs Backpack").
Ожидаемый результат: кнопка меняет название на "Remove", в иконке корзины (справа вверху) появляется цифра "1".
Фактический результат: кнопка меняет название на "Remove", в иконке корзины появляется цифра "1".

## TC-010: Удаление товара из корзины
Предусловия: пользователь авторизован (standard_user), в корзине уже есть один товар (счётчик = 1).
Шаги:

Ожидаемый результат: кнопка меняет название на "Add to cart", иконка корзины становится пустой (цифра исчезает).

## TC-011: Проверка фильтра "Price (low to high)"
Предусловия: пользователь авторизован (standard_user), находится на странице товаров.

## TC-012: Успешное оформление заказа (позитивный сценарий)
Предусловия: пользователь авторизован (standard_user), один товар добавлен в корзину.
Шаги:
1. Нажать на иконку корзины (переход на /cart.html).
2. Нажать кнопку "Checkout" (переход на /checkout-step-one.html).
3. Ввести "Иван" в поле "First Name".
4. Ввести "Иванов" в поле "Last Name".
5. Ввести "123456" в поле "Zip/Postal Code".
6. Нажать кнопку "Continue" (переход на /checkout-step-two.html).
7. Нажать кнопку "Finish".
Ожидаемый результат: происходит переход на страницу /checkout-complete.html, отображается зелёная галочка и сообщение об успешном оформлении заказа.
Фактический результат: происходит переход на /checkout-complete.html, отображается зелёная галочка.

## TC-020: Проверка выхода из системы (Logout)
